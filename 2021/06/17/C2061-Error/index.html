发表于 2021-06-17

Compiler Error C2601

简介

C2601报错提示：

  • “function”: 本地函数定义是非法的
  • 在C2601错误位置之前，source code可能有一个额外的{

ex:

分析原因

在main中定义了function函数，导致编译器无法识别{

踩过的坑：

合并代码之后出现了这个错误，查找官方文档如上所述

通过比对合并前后代码发现：前后源码文件编码方式出现了差异，主要为：

  • 字符编码不同
  • 换行符的编码不同

通过修改上述两种编码方式成功得到解决。

# C++
C2061 Error
vs2019修改字符编码方式
